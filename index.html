My setup

I like to setup my computer to my needs, but I also like to change hardware and software gears regularly. Certainly when I switch hardware, I tend to loose some of my software tweaks, so this section is an attempt to document my setup. Mainly for my own sake, but why not share it with the world.

OS

I use MacBook Pro (13-inch, 2017) and thus macOC Mojave, so software mentioned here is only tested on this platform
